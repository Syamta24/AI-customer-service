# 🤖 AI Telegram Customer Service Bot

## 🚀 Overview

This project is an AI-powered Telegram bot designed to automate customer service for online businesses.

It simulates a real human admin by responding to customer inquiries, recommending products, and assisting customers in making purchase decisions.

## ⚙️ Features

* AI-powered smart replies
* 24/7 automated customer service
* Human-like conversation
* Easy deployment on VPS

## 🧠 Tech Stack

* Node.js
* Telegram Bot API
* OpenAI API

## 📦 Installation

```bash
git clone https://github.com/Syamta24/telegram-ai-customer-service
cd telegram-ai-customer-service
npm install
```

## ▶️ Run

```bash
npm start
```

## 🔑 Setup

Rename `.env.example` to `.env` and fill in:

```
TELEGRAM_BOT_TOKEN=your_token
OPENAI_API_KEY=your_key
```

* Online shops
